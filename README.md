# Phoenix Finances
Aplicação para controle de gastos.

## Requisitos Funcionais
### Conta (Account)
- [ ] Criar uma conta
- [ ] Excluir uma conta
- [ ] Atualizar uma conta
- [ ] Realizar Login
- [ ] Realizar Autenticação

### Patrimônio (Equity)
- [ ] Patrimônio (Equity)
- [ ] Adicionar saldo
- [ ] remover saldo

### Transações (Transactions)
- [ ] Adicionar transação
- [ ] Mostrar histórico de transações

## Requitos não funcionais
### Conta (Account)
- [ ] Visualizar informações da conta
### Patrimônio (Equity)
- [ ] Opção de descrição ao adicionar ou remover saldo
- [ ] Visualizar o histórico do saldo
### Linguagem de programação
- [ ] Javascript

## Regras de negócio
- [ ] A conta deverá ser composta por email / senha
- [ ] Será permitido apenas 1 conta por email
